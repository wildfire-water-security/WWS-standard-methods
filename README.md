# QAQC-scripts
Scripts used to QAQC data streams generated within the project

## Instructions 
Any standardized scripts, directions, or other files related to performing QA/QC on data generated by the project should be stored here so others can easily find and use and we are processing all the data the same way across the project.

Please create a folder for each data type (NPOC, BPCA, DOM, Sonde-Data, etc.)
